# web-common

Web前端公共基础 JavaScript 库

## 技术栈

模块打包：rollup

测试：jest

代码质量：eslint

代码风格：prettier

Git Commit提交：husky + lint-staged + commitlint

## 开发

开发

`$ npm run dev`

构建代码

`$ npm run build`

校验 JS

`$ npm run lint`

登录 npm

`$ npm login`

发布包

`$ npm publish`

## 使用

安装

`$ npm install --save @zengbin/web-common`
